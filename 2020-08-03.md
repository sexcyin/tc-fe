- 选择排序
1. 首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置
2. 再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾
3. 重复第二步，直到所有元素均排序完毕
```javascript
function selectSort(arr) {
  for(let i = 0; i< arr.length; i++) {
    let min = arr[i];
    for (let j = i + 1; j< arr.length; j++) {
      if (arr[j] < min) {
        let temp = min;
        min = arr[j];
        arr[j] = temp;
      }
    }
    arr[i] = min;
  }
  return arr
}
```
